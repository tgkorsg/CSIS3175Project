Contributors: Juman Kim, Steve 
Android Gradle Plugin Version: 3.5.2

== How to Install == 
1. Open the android project
2. Click "Build" tab
3. Under "Build Bundle(s) / APK(s)", click "Build APK(s)

== Description ==
This app let users explore meme images from Imgur.com.
Meme images can be kept, downloaded, shared, and set as wallpaper.
Since this app does not contain advertisements and .gif files, users can save their data and more focus on it.
